Projeto acadêmico desenvolvido durante o Bacharelado em Ciência da Computação no Centro Universitário Campos de Andrade (UNIANDRADE). Trata-se da concepção e modelagem de um jogo de memória voltado para o público infantil (9 a 12 anos).

📋 Sobre o Projeto
O objetivo central foi implantar e modelar um jogo que estimula a memória através da repetição de sequências de cores e sons, utilizando a linguagem de programação C.
O desenvolvimento seguiu etapas rigorosas de:
Planejamento: Definição de cronograma e organização de atividades.
Modelagem: Elaboração de requisitos funcionais, não funcionais e fluxograma do sistema.
Implementação: Desenvolvimento do código-fonte na linguagem C.

⚙️ Especificações Técnicas
Linguagem de Programação: C.
Plataforma: Console/Terminal.
Recursos de Áudio: Sistema de "beeps" sonoros com frequências distintas para cada cor.

✨ Funcionalidades (Requisitos)Geração Aleatória: 
O sistema gera sequências de cores aleatoriamente a cada rodada.
Interface Sonora e Visual: Cada cor possui uma tecla correspondente e um som único.
Dificuldade Progressiva: A cada rodada, o tempo de duração dos sinais sonoros diminui em 100 milissegundos, aumentando o desafio.
Validação de Jogada: Se o jogador informar uma cor incorreta, o sistema encerra a partida imediatamente.
Ranking: Exibição da melhor marca do jogador após o fim de cada rodada.

🛠️ Cronograma de Desenvolvimento
O projeto foi executado em um ciclo ágil durante o ano de 2021:
Maio: Início do planejamento, organização e modelagem dos requisitos.
Junho: Elaboração do fluxograma e programação do código em C.
Finalização: Apresentação e defesa do projeto na disciplina de Lógica Matemática.
